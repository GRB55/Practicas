Archivo: `ANALISIS DE FORMULA1.ipynb`  
Herramientas:
- Numpy
- Pandas
- Matplotlib
- Seaborn

Descripción:
Este análisis explora datos históricos de Fórmula 1 para identificar tendencias y patrones relevantes.
